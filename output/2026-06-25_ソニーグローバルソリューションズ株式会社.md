## 📌 30秒サマリー

ソニーグループ株式会社の完全子会社（100%出資）として、グループ唯一のIT専門機能会社。ゲーム・音楽・映画・エレクトロニクスなど異なるビジネスセグメントを横断し、**グローバル約2,000名のITエンジニア**で200社以上のグループ企業のDX・AI・インフラを内側から支える。売上高439億円（FY2025）、**3期連続赤字から2025年3月期に黒字転換**。外部SIerにはない「立案段階からビジネスに寄り添う」提案型IT、Enterprise LLMなど最先端AI基盤の実装現場、グローバルな裁量環境が差別化ポイント。

**重要キーワード（5個）**: ビジネスイネーブラー / デジタルトランスフォーメーション（DX）/ グローバル / 変革・適応力 / 全体最適・Boundary Spanner

## 🏢 事業・ビジネスモデル

ソニーグループ唯一のIT専門機能会社として、グローバル約2,000名のITエンジニアが以下7領域でサービスを提供。売上のほぼ全てがグループ内取引（インターカンパニー）。

| 事業領域 | 内容 |
|---|---|
| ビジネスアカウント | グループ各事業のITニーズを吸い上げ、ソリューションを提案 |
| ビジネスプロセスデザイン | 次世代業務プロセス設計・RPA活用による業務効率化 |
| デジタルトランスフォーメーション | AI・機械学習を用いた新規ビジネス価値創造支援 |
| アプリケーション開発 | 製造・販売・人事・財務・SCMなど機能領域別のシステム構築 |
| アプリケーションマネジメント | 世界中で稼働するシステムの保守・運用 |
| インフラストラクチャーマネジメント | ITインフラ提供・クラウド化推進 |
| クオリティコントロール | プロジェクト品質管理・実行力強化支援 |

出典: [事業紹介 | ソニーGS公式](https://www.sonyglobalsolutions.jp/corporate/business/)

### 強み

- グループ事業への圧倒的な業務理解（立案段階から参画できる提案型IT）
- グローバル2,000名体制（米・英・シンガポール・中・印）
- Enterprise LLM・Sony Data Oceanなど最先端AI/データ基盤の実装現場
- ゲーム・音楽・映画・半導体と多業種横断の多様なプロジェクト経験
- ソニーグループ100%子会社として安定した事業基盤

### 弱み・リスク

- 売上ほぼ100%がグループ内取引（ソニーGの業績次第でリスク）
- 外販なし → 市場競争力・汎用スキルの評価軸が不明確
- FY2022〜FY2024の3期連続営業赤字（FY2025で黒字転換）
- 法人が異なるためソニー本体へのキャリア異動が制限される可能性（口コミ傾向）

### 中期的な注力領域

- **生成AI（Enterprise LLM）のグローバル展開**: グループ200社以上・4万5,000人へ展開、月200万回以上利用・**月5万時間削減**を実現（出典: [日経クロステック](https://xtech.nikkei.com/atcl/nxt/column/18/03251/062600004/)）
- **Sony Data Ocean**: グループ横断データ統合プラットフォームで「One Sony」戦略をIT面から支える
- **クラウドインフラモダナイゼーション**: AWS・Azure活用のクラウドシフト推進
- **DX内製化**: 外部委託から内製化への転換。ビジネスプロセス改革加速

## 📈 業績・財務

非上場企業（官報決算公告ベース）。IR開示なし。資本金1億円。

| 決算期 | 売上高 | 営業利益 | 純利益 |

出典: 官報決算DB [第38期](https://catr.jp/companies/70474/11817/settlements/35a45/411773) / [第37期](https://catr.jp/companies/70474/11817/settlements/0bd12/360778) / [第36期](https://catr.jp/settlements/d3b37/306417) / [第35期](https://catr.jp/companies/70474/11817/settlements/1a983/261609)

**トレンド**: 売上は3年間で約23%成長（357億→439億円）。FY2025に前期比+12.88%の大幅増収とともに**黒字転換**。利益剰余金がプラス（2億8,900万円）に転じた。

**面接で使える業績ポイント3つ**:
1. FY2025は売上+12.88%で3期連続赤字からの黒字転換——「コスト削減でなく売上成長による収益改善」が本質
2. 3年間で売上23%成長——親会社ソニーGのDX投資拡大とIT需要増が直接反映
3. ソニーGFY2026営業利益が**過去最高1兆4,475億円**（出典: [Business+IT](https://www.sbbit.jp/article/cont1/185323)）——IT投資縮小リスクは低く、SGSへの需要は拡大局面

## 🎤 面接対策

### 想定質問と回答のヒント

- **Q1. 最近のソニーのニュースで気になったものは？** → Enterprise LLM（月200万回利用・月5万時間削減）を引用。「その基盤を支えるSGSの役割に惹かれた」と接続。
- **Q2. ソニーグループの中でなぜSGSなのか？** → 「グループ横断の全体最適視点」「立案段階からビジネスに関与できる唯一の組織」を強調。
- **Q3. SGSの弱みだと思う点は？** → 外販なし指摘を認識のうえ「グループ最先端ビジネスの課題を直接扱える深さとグローバル規模での実装経験が中長期的な価値」と転換。
- **Q4. 海外勤務への意向は？** → グローバル展開を重視する会社のため頻出。具体的な志向と理由を準備。
- **Q5. ソニーFGのスピンオフをどう見るか？** → エンタメ・テクノロジー特化によりSGSの担う事業がより明確化・重要化すると論じる。
